# retro-snake-game-c
Retro Old School Basic Snake Game

# Dependencies
<ul>
  <li>Standard C libraries</li>
  <li>ncurses library
    <ul>
    <li>$ sudo apt-get install libncurses5-dev libncursesw5-dev</li>
    </ul>
  </li>
</ul>

# Compiling
`$ gcc -o snake snake.c -lncurses`

# Run
`$ ./snake`

# Controls
`Arrow Keys` to move snake
